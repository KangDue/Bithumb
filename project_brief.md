https://github.com/sharebook-kr/pybithumb 
pybithumb은 비트코인 거래소인 빗썸의 API를 쉽게 사용할 수 있도록 해주는 파이썬 라이브러리입니다.
이 라이브러리를 사용하면 비트코인 거래소의 API를 호출하여 거래, 잔고, 거래 내역 등을 확인하고 조작할 수 있습니다.
또한, 이 라이브러리는 비트코인 거래소의 API 문서를 참고하여 개발에 필요한 기능을 제공합니다.
이 라이브러리를 사용하면 비트코인 거래소의 API를 쉽게 호출하고, 개발에 필요한 기능을 빠르게 구현할 수 있습니다.

이 라이브러리와 github, 문서를 참조해서
자동거래, 거래전략 수립, 다양한 기본 전략, 백테스팅 등이 가능한
simple한 ui로 pyqt5와 pyqtgraph 기반, python 3.11 버전으로 프로그램 만들어줘.
사용자 api키 입력등 필수적인 부분도 빼먹지마
